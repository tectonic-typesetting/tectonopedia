\entry{primitive}{Primitive}{Primitive}
\DeclareTerm*{\TeX\ primitive}{TeX primitive}

A \b{primitive} is a basic command built into the \TeX\ engine.

Each primitive is associated with a specific control sequence. However, because
the meaning of a control sequence can be reassigned arbitrarily, the appearance
of a primitive's name in some \TeX\ source may not mean that that primitive is
actually invoked. A prominent example of this is the \tex`\end` control sequence
in \LaTeX, which is not the same thing as \e{end}. The control sequence
associated with a given primitive can be thought of as its name in an abstract
sense, but in any given document that control sequence may have a different
meaning.

The primitives can be grouped into several broad categories:

\begin{itemize}
  \item 80 integer parameters,
  \item 23 dimensional parameters,
  \item 19 glue parameters (16 regular and 3 math),
  \item 12 token-list parameters,
  \item 5 shape or penalties parameters
  \item 361 other primitives.
\end{itemize}
